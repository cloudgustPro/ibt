terminal commands

1. ls: used for listing the directories and files in that folder.
2. mkdir [folder_name]: used to create directory.
3. cd [folder]: moves you into a specific folder.
4. cd ~: returns to in to home (root) folder instantly.

git commands

1. git switch [branch_name]: used to change one branch to another branch OR change main branch to the other branch
2. git checkout -b [branch_name]: used to create new branch rather than the default(main)
3. git pull: downloads changes from the remote repo and integrate them into current local folder
